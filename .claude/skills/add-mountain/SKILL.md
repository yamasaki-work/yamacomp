### フォーマットルール
- エリア見出し（`## ○○エリア`）が既存なら、そのセクション末尾に追加する
- エリアが新規なら、ファイル末尾に新しいセクションを作る
- 各エントリは以下の形式を厳守する：

```
### 山名
- 難易度：初級 / 初級〜中級 / 中級 / 中級〜上級 / 上級 のいずれか
- 歩行時間：○〜○時間（往復）
- 特徴：（100〜150字程度。見どころ・特徴・注意点を含める）
- アクセス（電車）：（最寄り駅からのルート）
- アクセス（車）：（最寄りICからのルート）
```

- 情報が不足している項目は「要確認」と記入する
- 火山・危険地帯の場合は特徴に「必ず最新情報を確認すること」を付記する
- 追加後、追加した山名とエリアを報告する
